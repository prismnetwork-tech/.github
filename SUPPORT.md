# Support

Community support is provided through repository discussions and issues on a
best-effort basis. The open-source projects do not provide uptime,
response-time or compatibility service-level agreements.

Use:

- Discussions for deployment and integration questions.
- Issues for reproducible bugs and scoped feature proposals.
- Private vulnerability reporting for security defects.

Include versions, operating system, sanitized configuration and a minimal
reproduction. Never include credentials, private keys, wallet secrets, session
tokens or user data.
