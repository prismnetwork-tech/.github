# Prism Network organization configuration

This repository publishes the Prism Network organization profile and provides
default community-health files for repositories that do not define their own.

The rendered organization profile lives in [`profile/README.md`](profile/README.md).

## Scope

- Organization profile and project directory.
- Default contribution, support and security policies.
- Default issue and pull-request templates.
- Validation for prohibited credentials, personal paths and attribution.

Repository-specific policies take precedence when a project carries materially
different security or release requirements.
